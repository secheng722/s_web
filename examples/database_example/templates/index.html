📊 Database Example - User Management API

使用 SQLite 数据库的完整 CRUD API 示例

📋 API 端点

GET /api/v1/users

获取所有用户列表

curl http://127.0.0.1:3000/api/v1/users
POST /api/v1/users

创建新用户

curl -X POST http://127.0.0.1:3000/api/v1/users \ -H 'Content-Type: application/json' \ -d '{"name":"张三","email":"zhangsan@example.com"}'
GET /api/v1/users/:id

获取特定用户信息

curl http://127.0.0.1:3000/api/v1/users/{user_id}
PUT /api/v1/users/:id

更新用户信息

curl -X PUT http://127.0.0.1:3000/api/v1/users/{user_id} \ -H 'Content-Type: application/json' \ -d '{"name":"李四","email":"lisi@example.com"}'
DELETE /api/v1/users/:id

删除用户

curl -X DELETE http://127.0.0.1:3000/api/v1/users/{user_id}
GET /health

健康检查

curl http://127.0.0.1:3000/health

📊 数据模型

{ "id": "uuid", "name": "string", "email": "string", "created_at": "ISO 8601 timestamp" }

🚀 快速开始

  1. 确保已安装 Rust 和 Cargo
  2. 运行示例：cargo run
  3. 访问 API 或使用上面的 curl 命令测试
